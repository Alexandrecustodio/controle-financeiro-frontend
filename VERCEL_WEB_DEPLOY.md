## 🎯 Passo-a-Passo (5 Minutos)

### **Passo 1: Criar Conta Vercel**

1. Acesse https://vercel.com
2. Clique **"Sign Up"**
3. Escolha **"Continue with GitHub"**
4. Autorize Vercel
5. Pronto! Conta criada

---

### **Passo 2: Importar Projeto**

1. Na dashboard Vercel, clique **"Add New..."** → **"Project"**
2. Clique **"Import Git Repository"**
3. Procure por **`controle-financeiro-frontend`**
4. Clique **"Import"**

---

### **Passo 3: Configurar Variáveis de Ambiente**

1. Na página de configuração, procure **"Environment Variables"**
2. Adicione:
   ```
   EXPO_PUBLIC_API_URL = https://seu-backend.railway.app/api
   ```
   (Substitua pela URL do seu backend no Railway)

3. Clique **"Deploy"**

---

### **Passo 4: Aguardar Deploy**

O Vercel vai:
1. Clonar seu repositório
2. Instalar dependências
3. Compilar o projeto
4. Fazer upload para servidores

**Tempo:** 2-5 minutos

Você verá:
```
✓ Build successful
✓ Deployment complete
```

---

### **Passo 5: Acessar Seu App**

Vercel vai gerar uma URL como:
```
https://controle-financeiro-frontend.vercel.app
```

**Abra no navegador do seu iPhone/Android e pronto!** 🎉

---

## 🆘 Troubleshooting

### "Build failed"
- Verifique se o código está correto no GitHub
- Verifique se todas as dependências estão instaladas
- Veja os logs de erro no Vercel

### "App não carrega"
- Verifique se a URL do backend está correta
- Verifique se o backend está rodando
- Limpe o cache do navegador (Ctrl+Shift+Delete)

### "Dados não sincronizam"
- Verifique a URL da API em "Environment Variables"
- Verifique se o backend está acessível
- Abra o console do navegador (F12) para ver erros
